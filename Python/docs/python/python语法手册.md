## 五、各章小白易错点速览（查漏用）

- **01 变量**：`list1 = list2 = []` 会共用一个列表；解包时等号两边个数要对齐。
- **02 数据结构**：`.sort()` 改原列表且返回 `None`，别写 `a = lst.sort()`；`.pop(i)` 按索引、`.remove(x)` 按值；字典 `in` 默认查键不查值。
- **03 异常**：`except` 后尽量写具体异常类型；`else` 是 try 没异常时执行，`finally` 无论如何都执行。
- **04 格式化**：f-string 里自定义填充字符必须写对齐方式（如 `1>`），否则会被当成宽度；`print(obj)` 用 `__str__`，容器里显示用 `__repr__`。
- **05 内置函数**：`max`/`min` 按规则比用 `key=lambda x: x["字段"]`；`sorted()` 不改原数据，`.sort()` 只用于列表且改原数据。
- **06 逻辑与真值**：`and` 返第一个假或最后一个，`or` 返第一个真或最后一个；假值除 `None`/`False`/0 外还有空串、空列表、空字典等。
- **07 身份与成员**：`==` 比内容，`is` 比内存地址；`in` 在列表/字符串是 O(n)，在集合/字典键是 O(1)。
- **08 数据处理**：进制换算要校验范围（如 2–16）；单位换算注意除数不为 0、参数为正数。
- **09 字符编码**：编码用 `.encode()`，解码用 `.decode()`；`ord('中')` 是码点，`chr(65)` 是字符。
- **10 循环**：`range(stop)` 不包含 stop；`break` 只跳出一层循环，`return` 直接结束函数。

更细的题目与要点见 [错题本](/python/Python核心语法-错题本)。
