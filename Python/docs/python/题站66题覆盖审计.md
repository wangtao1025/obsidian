## 四、部分覆盖题的“具体缺口”

### 4.1 题 6 数据类型

当前缺口：
- 缺一个真正的“全局数据类型总览”
- `bool`、`NoneType`、`bytes` 没有和其他类型一起做统一心智图
- 缺少“按可变 / 不可变、序列 / 映射 / 集合、文本 / 字节”这种更上层的分类

最合适处理方式：
- 在当前手册最前面新增一章或 0 章《数据类型总览》

### 4.2 题 42 浅拷贝与深拷贝

当前缺口：
- 缺最关键的嵌套列表联动示例
- 缺“什么时候浅拷贝就够用，什么时候必须深拷贝”
- 缺和赋值共享引用的串联

最合适处理方式：
- 继续扩充 `python语法手册-11-标准库.md` 的 `copy` 小节

### 4.3 题 53 Pickle

当前缺口：
- 缺 `dump/load` 文件版 API
- 缺“适合内部 Python 对象持久化，不适合不可信输入”的安全提醒
- 缺文本格式与二进制序列化的对比语境

最合适处理方式：
- 扩充 `python语法手册-11-标准库.md` 的 `pickle` 小节

### 4.4 题 58 装饰器

当前缺口：
- 缺 `@decorator` 语法糖展开解释
- 缺通用 wrapper 模板
- 缺 `*args/**kwargs` 说明
- 缺 `functools.wraps`
- 缺带参数装饰器

最合适处理方式：
- 扩充 `python语法手册-函数与作用域.md`
- 如果内容再变大，可以拆单篇《装饰器与高阶函数》

### 4.5 题 60 标准库模块
