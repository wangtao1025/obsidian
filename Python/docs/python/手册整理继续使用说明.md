## 五、当前这套整理风格

每章默认采用下面这套结构：

### 主线层
- 一句话先懂
- 标准语法 / 核心规则
- 最小例子
- 易错点 / 开发建议

### 展开层
只在复杂知识点处展开：
- 为什么会这样
- 常见误区
- 小白向逐步拆解
- 但不能整章都写得太口语化，避免影响专业读者检索

### 额外要求
- 不要整体搬运题站
- 不要写“面试题站补充”这类字样
- 保证右侧导航能正常显示关键标题
- 完成每章后，要回看自测卷覆盖和题站知识点覆盖
