## 1. 多进制数字

在 Python 中，除了十进制外，还支持二进制、八进制和十六进制。掌握多进制表示与转换对位运算、颜色代码、硬件交互等场景很重要。

Python 用**前缀**区分不同进制：

| 进制     | 前缀      | 示例     | 十进制值 |
|----------|-----------|----------|----------|
| 二进制   | `0b` 或 `0B` | `0b1010` | 10       |
| 八进制   | `0o` 或 `0O` | `0o12`   | 10       |
| 十六进制 | `0x` 或 `0X` | `0xA`    | 10       |

---

## 2. 基本表示方法

### 2.1 直接表示不同进制的数字

在代码里写带前缀的字面量，得到的就是**整数**，可直接参与运算。

```python
# 二进制 1010 = 十进制 10
binary_num = 0b1010
octal_num = 0o12      # 八进制 12 = 10
hexadecimal_num = 0xA # 十六进制 A = 10

print(f"0b1010 = {binary_num}, 0o12 = {octal_num}, 0xA = {hexadecimal_num}")
print(binary_num == octal_num == hexadecimal_num)  # True

# 较大数：0b11111111 = 0o377 = 0xFF = 255
print(0b11111111, 0o377, 0xFF)  # 255 255 255

# 负数：前缀前加负号
print(-0b1010, -0o12, -0xA)  # -10 -10 -10
```

### 2.2 字符串形式表示

用**字符串**写 "1010"、"FF" 等时，只是文本，不能直接做数学运算，需要先用 `int(字符串, 基数)` 转成整数。

```python
binary_str = "1010"
hex_str = "A"
print(type(binary_str))   # <class 'str'>
print(type(0b1010))       # <class 'int'>
# 运算前先转换：int("1010", 2) → 10
```

## 3. 进制转换

### 3.1 将其他进制转换为十进制

用 **`int(字符串, base)`**：第一个参数是**字符串**，第二个是**该字符串所在的进制**（2～36）。返回十进制整数。

```python
# 字符串 + 基数 → 十进制整数
print(int("1010", 2))   # 10
print(int("12", 8))     # 10
print(int("A", 16))     # 10
print(int("FF", 16))    # 255

# 负数字符串也可转换
print(int("-1010", 2))  # -10

# 任意进制（2～36）：如 5 进制 "123"
print(int("123", 5))    # 38 = 1*25 + 2*5 + 3
```

### 3.2 将十进制转换为其他进制

用内置函数得到**带前缀的字符串**：

| 函数         | 作用           | 返回值示例        |
|--------------|----------------|-------------------|
| `bin(number)` | 十进制 → 二进制 | `'0b1010'`        |
| `oct(number)` | 十进制 → 八进制 | `'0o12'`          |
| `hex(number)` | 十进制 → 十六进制 | `'0xa'` / `'0xFF'` |

```python
n = 10
print(bin(n))   # 0b1010
print(oct(n))   # 0o12
print(hex(n))   # 0xa

n = 255
print(bin(n), oct(n), hex(n))  # 0b11111111 0o377 0xff

# 负数会得到负号 + 绝对值的进制表示
print(bin(-10))  # -0b1010

# 去掉前缀：切片 [2:]
print(bin(10)[2:])   # 1010
print(hex(255)[2:])  # ff
```

### 3.3 任意进制之间的转换

思路：**先转十进制，再转目标进制**。其他进制 → 十进制用 `int(str, base)`；十进制 → 二/八/十六用 `bin`/`oct`/`hex`；十进制 → 其他进制用**除基取余**自己实现。

#### 3.3.1 自定义进制转换函数

```python
def convert_to_custom_base(decimal, base):
    """将十进制数转换为自定义进制字符串（2～36）。"""
    if decimal == 0:
        return "0"
    digits = "0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZ"
    result = ""
    n = abs(decimal)
    while n > 0:
        result = digits[n % base] + result
        n //= base
    return "-" + result if decimal < 0 else result

def convert_base(number, from_base, to_base):
    """将数字从 from_base 进制转为 to_base 进制（number 为字符串）。"""
    decimal = int(str(number), from_base)
    if to_base == 2:
        return bin(decimal)
    if to_base == 8:
        return oct(decimal)
    if to_base == 16:
        return hex(decimal)
    return convert_to_custom_base(decimal, to_base)

# 示例
print(convert_base("1010", 2, 8))    # 0o12
print(convert_base("1010", 2, 16))  # 0xa
print(convert_to_custom_base(10, 5))  # 20（10 的 5 进制）
```

#### 3.3.2 进制转换工具类

```python
class BaseConverter:
    def __init__(self):
        self.digits = "0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZ"

    def to_decimal(self, number, base):
        """任意进制字符串 → 十进制整数"""
        return int(str(number), base)

    def from_decimal(self, decimal, base):
        """十进制整数 → 任意进制字符串"""
        if decimal == 0:
            return "0"
        result = ""
        n = abs(decimal)
        while n > 0:
            result = self.digits[n % base] + result
            n //= base
        return "-" + result if decimal < 0 else result

    def convert(self, number, from_base, to_base):
        """任意进制 → 任意进制（经十进制中转）"""
        decimal = self.to_decimal(number, from_base)
        return self.from_decimal(decimal, to_base)

converter = BaseConverter()
print(converter.to_decimal("1010", 2))      # 10
print(converter.from_decimal(10, 5))        # 20
print(converter.convert("1010", 2, 5))     # 20
```

## 4. 总结

- **表示**：二进制 `0b`、八进制 `0o`、十六进制 `0x`；带前缀的是整数，可直接运算。
- **转十进制**：`int(字符串, base)`，base 为 2～36。
- **十进制转 2/8/16**：`bin()` / `oct()` / `hex()`，得到带前缀的字符串；去前缀用 `[2:]`。
- **任意进制互转**：先 `int(..., from_base)` 得十进制，再 `bin`/`oct`/`hex` 或除基取余得到目标进制。
- **注意**：字符串形式的数字不能直接运算；负数转换注意符号；自定义进制时需校验 base 与字符合法性。

## 5. 参考回答（面试向）

### 5.1 基本回答

- **表示**：二进制 `0b`、八进制 `0o`、十六进制 `0x`，如 `0b1010`、`0xA` 都表示 10。
- **转十进制**：`int("1010", 2)`、`int("FF", 16)` 等。
- **十进制转其他**：`bin()`、`oct()`、`hex()` 返回带前缀的字符串；任意进制可先转十进制再用除基取余实现。
- **应用**：二进制常用于位运算与硬件；八进制常见于 Unix 权限；十六进制常用于颜色、内存地址、协议。
- **技巧**：`int()` 支持 2～36 进制；去掉前缀用切片 `bin(10)[2:]`。

### 5.2 加分要点

- 项目经验：颜色值、文件权限、像素值等用多进制表示更直观。
- 性能：直接写 `0xFF` 比 `int("FF", 16)` 在解析阶段更省一步；大量转换可先转成整数再处理。
- 扩展：加密、压缩、协议解析里常涉及进制与位运算。

### 5.3 可能的追问

- **为何用不同进制？** 二进制适合位运算，八进制适合权限，十六进制适合字节与可读性折中。
- **任意进制怎么实现？** 十进制作桥梁：先 `int(s, from_base)`，再除基取余得到目标进制字符串。
- **注意点？** 输入需符合该进制字符集（如二进制只能 0/1）；进制范围 2～36；负数符号单独处理。

### 5.4 回答技巧

先说三种进制与前缀 → 再说 `int` 与 `bin`/`oct`/`hex` → 举一两个例子 → 提应用与注意事项。

### 5.5 高级加分点

- **int 支持 2～36 进制**，超过 10 用 A～Z 表示。
- **错误处理**：校验进制范围与字符合法性，非法输入会触发 `ValueError`。
- **与位运算结合**：进制转换常和 `&`、`|`、移位一起用于掩码、标志位等。
