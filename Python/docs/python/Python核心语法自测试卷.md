## 一、变量赋值与解包

**1.1** `a, b, c = 1, 2, 3`

**1.2** 所有变量指向同一个内存地址（同一个列表对象）。修改其中一个，其他全部跟着变。原因：列表是可变对象，链式赋值让所有变量引用同一个对象。

**1.3** `list_a, list_b = [], []`

**1.4** `[2, 3, 4]`

**1.5** `{'a': 1, 'b': 3, 'c': 4}`（后面的字典覆盖前面的同名键）

**1.6** 输出 `False`。隐患：字典是可变对象，`config_a` 和 `config_b` 指向同一个字典。修复：`config_a, config_b = {"debug": True}, {"debug": True}`

**1.7** 会抛出 `ValueError`（解包时元素数量与变量个数不匹配）。`a, b = [1, 2, 3]` 为“值太多”；`a, b, c = [1, 2]` 为“值太少”。

## 二、赋值与算术运算符

**2.1** 简单赋值运算符是 `=`。两种写法：① `a, b, c = 10, 20, 30`（多重赋值）；② `data = [10, 20, 30]`；`a, b, c = data`（序列解包）。

**2.2** `+=` 等价于 `x = x + y`，其余类推：`-=`、`*=`、`/=`、`//=`、`%=`、`**=` 分别为减、乘、除、整除、取余、幂后再赋值。例：`x += 5` 即 `x = x + 5`。

**2.3** 依次为：`10` → `15` → `12` → `24` → `6.0`。

**2.4** 计数器：`counter += 1`；累加器：`total += num`；字符串拼接：`message += " World"`；列表扩展：`my_list += [4, 5]` 或 `my_list *= 2`。

**2.5** 加 `+`、减 `-`、乘 `*`、除 `/`、整除 `//`、取余 `%`、幂 `**`。整除为向下取整，取余为求余数，幂为次方。

**2.6** `10 / 3` 结果为浮点型 `3.333...`；`10 // 3` 为 `3`；`10 % 3` 为 `1`。

**2.7** `2 ** 3` 为 `8`；`2 ** 3 ** 2` 等价于 `2 ** (3 ** 2) = 2 ** 9 = 512`（幂右结合）；`-10 // 3` 为 `-4`；`-10 % 3` 为 `2`。

**2.8** ① 除零会抛出 `ZeroDivisionError`，需 try-except；② 浮点运算有精度误差，比较用 `math.isclose()`；③ 负数整除/取余在不同语言中规则不同，Python 中余数符号与被除数一致等。

**2.9** `hours = total_seconds // 3600`；`minutes = (total_seconds % 3600) // 60`；`seconds = total_seconds % 60`。

**2.10** 从高到低：幂 `**` → 一元 `+x -x ~x` → 乘除取余 `* / // %` → 加减 `+ -` → 比较 → `not` → `and` → `or` → 赋值。口诀：算术 > 比较 > 逻辑 > 赋值；幂最高，赋值最低。

**2.11** 先算 `c ** 2`，再 `b * (c**2)`，最后 `a + ...`。先算 `a + b` 再乘：`(a + b) * (c ** 2)`。

**2.12** 结合性指同优先级时从左到右或从右到左计算。`**` 右结合，故 `2 ** 3 ** 2 = 2 ** (3**2) = 512`。赋值链从右向左：先 `c = 1`，再 `b = c`，再 `a = b`。

**2.13** 输出 `16`。先算 `2 * 3 = 6`，再执行 `d = d + 6`，即 `10 + 6 = 16`（复合赋值在算术之后执行）。

**2.14** 括号能明确计算顺序，避免因记错优先级导致逻辑错误，并提升可读性、便于他人维护。

**2.15** 先执行比较运算（`score > 60` 和 `team == "A"`），再执行逻辑与 `and`。比较优先级高于逻辑。

**2.16** 抛出 `ZeroDivisionError`。可用 `result = a / b if b != 0 else None` 或 try-except 捕获后赋 `None`。

**2.17** 浮点数用二进制近似存储，部分十进制小数无法精确表示。推荐用 `math.isclose(a, b)` 或 `abs(a - b) < 1e-9` 判断"足够接近"。

**2.18** 圆面积：`area = pi * r ** 2`。欧氏距离：`distance = ((x2-x1)**2 + (y2-y1)**2)**0.5`。

**2.19** `my_list += [4, 5]` 原地修改列表，`my_list` 仍指向原对象。`my_list = my_list + [4, 5]` 会创建新列表并赋给 `my_list`，若其它引用仍指向原列表则不受影响。

## 二附、位运算符

**二附.1** 六种：`&` 按位与（对应位都为 1 结果为 1）、`|` 按位或（有 1 则 1）、`^` 按位异或（不同为 1）、`~` 按位取反（所有位取反）、`<<` 左移（低位补 0）、`>>` 右移（高位按规则补）。  
**二附.2** `12 & 10` = 8，`12 | 10` = 14，`12 ^ 10` = 6，`~12` = -13。Python 中 `~x` 等价于 `-x - 1`。  
**二附.3** 左移 n 位等价于乘 2^n，右移 n 位等价于整除 2^n。`12 << 2` = 48，`12 >> 2` = 3。  
**二附.4** 正数右移很多位会变成 0（相当于一直整除 2）。负数不会变成 0，Python 是算术右移，会趋向 -1。  
**二附.5** `(n & 1) == 0` 为偶数，`(n & 1) == 1` 为奇数。原理：二进制最低位为 0 即偶数，为 1 即奇数，与 1 做按位与即得最低位。  
**二附.6** `1 << n` 即 2 的 n 次方。例：`1 << 5` = 32。  
**二附.7** `a ^= b`；`b ^= a`；`a ^= b`。（或 `a, b = a ^ b, a` 等等价写法）  
**二附.8** 添加权限：`current | permission`（按位或）；移除权限：`current & ~permission`（与取反后的掩码与）；检查权限：`(current & permission) == permission`（按位与后判断）。  
**二附.9** 位运算对整数的每一位操作，返回整数；逻辑运算对表达式真假求值，返回布尔或操作数（and/or 可返回操作数）。  
**二附.10** `0 10 20 5`（x&1=0 偶，x|2=10|2=10，x<<1=20，x>>1=5）。  
**二附.11** 不会。Python 整数无固定位宽，左移很多位只会得到很大的整数。C/Java 等有位数限制会溢出。  
**二附.12** `x ^ 0 == x`（与 0 异或等于自身）；`x ^ x == 0`（与自身异或等于 0）。

## 十一、循环遍历

**11.1** `for` 用于遍历已知序列（"数完"/"走完"）；`while` 用于条件循环（"等到"某个条件不满足）。

**11.2** `range(stop)`、`range(start, stop)`、`range(start, stop, step)`

**11.3** 不是列表，是惰性的 range 对象。不在内存中存储所有数字，按需生成，所以即使 `range(10**9)` 也只占很少内存。

**11.4** `list(range(5, 0, -1))`

**11.5**

| 关键字 | 作用范围 | 形象理解 |
|--------|---------|---------|
| `pass` | 无影响 | "此处招租"，占位防报错 |
| `continue` | 当前这一次迭代 | "这题跳过"，进入下一轮 |
| `break` | 最近的一层循环 | "原地爆破"，跳出循环 |
| `return` | 整个函数 | "终结者"，直接关掉函数 |

**11.6** `0 1 3 `（i=2 被 continue 跳过，i=4 被 break 终止）

**11.7** 只能跳出最近的一层循环。要跳出所有嵌套循环应该用 `return`（将循环放在函数中）。

**11.8** 支持。`list(range(10)[2:8:2])` → `[2, 4, 6]`

## 十二、标准库模块

**12.1** `datetime`（日期+时间）、`date`（仅日期）、`time`（仅时间）、`timedelta`（时间间隔）

**12.2**
```python
from datetime import datetime
now = datetime.now()
print(now.strftime("%Y-%m-%d %H:%M:%S"))
```

**12.3** `strptime`：字符串 → 时间对象（parse）；`strftime`：时间对象 → 字符串（format）。

**12.4**
```python
from datetime import datetime, timedelta
tomorrow = datetime.now() + timedelta(days=1)
```

**12.5** ① 可通过属性名访问（可读性好）② 仍支持索引访问 ③ 不可变 ④ 比普通类更省内存 ⑤ 自描述性强

**12.6**
```python
from collections import namedtuple
Point = namedtuple('Point', ['x', 'y'])
p = Point(10, 20)
```

**12.7** 替换指定字段的值并返回新对象，不会修改原对象（因为元组不可变）。
