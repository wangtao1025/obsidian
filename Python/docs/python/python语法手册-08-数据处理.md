## 一、数据处理

核心直觉： 算法是灵魂，单位是底线。
### 1.1 进制转换的“万能公式”

在 Python 中，处理进制转换只需记住一个中转站：**十进制**。

- **第一步：统一化**：利用 `int(str, base)` 将源数据（2-16进制）全部吞掉，转化成 Python 内部的十进制整数。

- **第二步：多样化**：
  
  - 如果转回 2/8/16 进制：直接使用内置函数 `bin()` / `oct()` / `hex()`。
  
  - 如果转成自定义进制（如 7 进制）：使用**循环取余法**（不断用 `//` 和 `%` 剥离每一位数字）。

```Python
# 核心逻辑示例 (十进制作为中转)
decimal_val = int("FF", 16)      # FF -> 255
binary_str = bin(decimal_val)    # 255 -> '0b11111111'
```

**多进制表示、转换函数与面试要点**见下 **二、进制表示与转换**。

### 1.2 存储单位格式化 (Human Readable)（自测卷 9.3 待复习）

为了让文件大小变得“人类可读”，我们需要在 **1024 进制**（KiB/MiB 系列）下进行单位降级。

- **核心算法**：使用 `while` 循环判定。只要数字 $\ge 1024$，就除以 1024.0 并且将单位列表索引 `+1`。

- **单位阶梯**：`['B', 'KiB', 'MiB', 'GiB', 'TiB']`。

- **全栈应用**：后端返回原始字节数 `1536`，前端通过此算法显示 `1.50 KiB`。

### 1.3 网络下载时间计算 (The 8-Bit Trap)

这是最容易被面试官抓住的细节陷阱：**大 B (Byte) 与 小 b (bit)**。

- **换算底线**：文件大小通常以 **MB** (Megabyte) 存储，而带宽速度以 **Mbps** (Megabit per second) 计算。

- **换算关键**：$1\text{ Byte} = 8\text{ bits}$。

- **避坑公式**：$\text{下载秒数} = (\text{文件 MB} \times 8) \div \text{带宽 Mbps}$。

- **逻辑分支**：根据总秒数判断输出单位（秒/分钟/小时），并使用 f-string 控制小数位数（如 `{val:.1f}`）。

### 1.4 健壮性检查 (Defensive Programming)

作为“谨慎的开发者”，函数的第一行永远应该是**边界检查**：

- **进制范围**：必须在 2 到 16 之间。

- **数值合法性**：文件大小不能为负数，带宽必须大于 0。

## 二、进制表示与转换 {#进制表示与转换}

在 Python 中，除了十进制外，还支持二进制、八进制和十六进制。掌握多进制表示与转换对位运算、颜色代码、硬件交互等场景很重要。

Python 用**前缀**区分不同进制：

| 进制     | 前缀      | 示例     | 十进制值 |
|----------|-----------|----------|----------|
| 二进制   | `0b` 或 `0B` | `0b1010` | 10       |
| 八进制   | `0o` 或 `0O` | `0o12`   | 10       |
| 十六进制 | `0x` 或 `0X` | `0xA`    | 10       |

### 2.1 基本表示方法

#### 直接表示不同进制的数字

在代码里写带前缀的字面量，得到的就是**整数**，可直接参与运算。

```python
# 二进制 1010 = 十进制 10
binary_num = 0b1010
octal_num = 0o12      # 八进制 12 = 10
hexadecimal_num = 0xA # 十六进制 A = 10

print(f"0b1010 = {binary_num}, 0o12 = {octal_num}, 0xA = {hexadecimal_num}")
print(binary_num == octal_num == hexadecimal_num)  # True

# 较大数：0b11111111 = 0o377 = 0xFF = 255
print(0b11111111, 0o377, 0xFF)  # 255 255 255

# 负数：前缀前加负号
print(-0b1010, -0o12, -0xA)  # -10 -10 -10
```

#### 字符串形式表示

用**字符串**写 "1010"、"FF" 等时，只是文本，不能直接做数学运算，需要先用 `int(字符串, 基数)` 转成整数。

```python
binary_str = "1010"
hex_str = "A"
print(type(binary_str))   # <class 'str'>
print(type(0b1010))       # <class 'int'>
# 运算前先转换：int("1010", 2) → 10
```

### 2.2 进制转换

#### 将其他进制转换为十进制

用 **`int(字符串, base)`**：第一个参数是**字符串**，第二个是**该字符串所在的进制**（2～36）。返回十进制整数。

```python
# 字符串 + 基数 → 十进制整数
print(int("1010", 2))   # 10
print(int("12", 8))     # 10
print(int("A", 16))     # 10
print(int("FF", 16))    # 255

# 负数字符串也可转换
print(int("-1010", 2))  # -10

# 任意进制（2～36）：如 5 进制 "123"
print(int("123", 5))    # 38 = 1*25 + 2*5 + 3
```

#### 将十进制转换为其他进制

用内置函数得到**带前缀的字符串**：

| 函数         | 作用           | 返回值示例        |
|--------------|----------------|-------------------|
| `bin(number)` | 十进制 → 二进制 | `'0b1010'`        |
| `oct(number)` | 十进制 → 八进制 | `'0o12'`          |
| `hex(number)` | 十进制 → 十六进制 | `'0xa'` / `'0xFF'` |

```python
n = 10
print(bin(n))   # 0b1010
print(oct(n))   # 0o12
print(hex(n))   # 0xa

n = 255
print(bin(n), oct(n), hex(n))  # 0b11111111 0o377 0xff

# 负数会得到负号 + 绝对值的进制表示
print(bin(-10))  # -0b1010

# 去掉前缀：切片 [2:]
print(bin(10)[2:])   # 1010
print(hex(255)[2:])  # ff
```

#### 任意进制之间的转换

思路：**先转十进制，再转目标进制**。其他进制 → 十进制用 `int(str, base)`；十进制 → 二/八/十六用 `bin`/`oct`/`hex`；十进制 → 其他进制用**除基取余**自己实现。

**自定义进制转换函数**：先按源进制转成十进制，再按目标进制“除基取余”得到结果。十进制 → 二/八/十六可直接用内置函数；→ 其他进制用自定义函数。

```python
# 字符表：余数 0～9 用数字，10～35 用 A～Z（支持 2～36 进制）
DIGITS = "0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZ"

def convert_to_custom_base(decimal, base):
    """将十进制数转换为自定义进制字符串（2～36）。支持负数。"""
    if decimal == 0:
        return "0"
    result = ""
    n = abs(decimal)   # 先按绝对值做除基取余，负数最后补负号
    while n > 0:
        remainder = n % base
        result = DIGITS[remainder] + result   # 当前余数对应字符拼到结果左侧
        n //= base
    return "-" + result if decimal < 0 else result

def convert_base(number, from_base, to_base):
    """将数字从 from_base 进制转为 to_base 进制。number 可为字符串或整数。"""
    decimal = int(str(number), from_base)
    if to_base == 2:
        return bin(decimal)
    if to_base == 8:
        return oct(decimal)
    if to_base == 16:
        return hex(decimal)
    return convert_to_custom_base(decimal, to_base)

# 示例
print(convert_base("1010", 2, 8))     # 0o12（二进制 1010 → 十进制 10 → 八进制 12）
print(convert_base("1010", 2, 16))    # 0xa
print(convert_to_custom_base(10, 5))  # 20（10 的 5 进制）
print(convert_to_custom_base(-10, 2)) # -1010（负数先按绝对值算，最后加负号）
```

### 2.3 进制与转换小结

- **表示**：二进制 `0b`、八进制 `0o`、十六进制 `0x`；带前缀的是整数，可直接运算。
- **转十进制**：`int(字符串, base)`，base 为 2～36。
- **十进制转 2/8/16**：`bin()` / `oct()` / `hex()`，得到带前缀的字符串；去前缀用 `[2:]`。
- **任意进制互转**：先 `int(..., from_base)` 得十进制，再 `bin`/`oct`/`hex` 或除基取余得到目标进制。
- **注意**：字符串形式的数字不能直接运算；负数转换注意符号；自定义进制时需校验 base 与字符合法性。

### 2.4 参考回答（面试向）

- **表示**：二进制 `0b`、八进制 `0o`、十六进制 `0x`，如 `0b1010`、`0xA` 都表示 10。
- **转十进制**：`int("1010", 2)`、`int("FF", 16)` 等。
- **十进制转其他**：`bin()`、`oct()`、`hex()` 返回带前缀的字符串；任意进制可先转十进制再用除基取余实现。
- **应用**：二进制常用于位运算与硬件；八进制常见于 Unix 权限；十六进制常用于颜色、内存地址、协议。
- **技巧**：`int()` 支持 2～36 进制；去掉前缀用切片 `bin(10)[2:]`。
